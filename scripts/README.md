# Scripts

Carpeta para almacenar scripts de automatización y utilidades del proyecto External ESP Free Fire.

## Contenido

- **build.sh**: Script para compilar el proyecto
- **deploy.sh**: Script para desplegar
- **[Añade tus scripts aquí]**

## Uso

```bash
./scripts/build.sh
./scripts/deploy.sh
```

Asegúrate de dar permisos de ejecución:
```bash
chmod +x scripts/*.sh
```
